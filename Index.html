Inici

A travers d'aquesta pàgina de prova, que és l'index.html de ProvaWeb (la primera web que feim a l'assignatura d'Aplicacions Web), veurem distintes coses.

Estructura d'una web

El codi d'una web (com a conjunt de distintes pàgines) es guarda dins una carpeta amb el nom de la web.

Tendrà els següents directoris:

estil: té codi amb css per donar estil i "pintar" les nostres webs

imatges: té les imatges de la nostra web.

pagines: té el codi html de totes les pàgines de la nostra web.

També hi trobarem el fitxer index.html "a lloure". És la pàgina d'inici o home de la nostra web

Estructura d'un document HTML

Un fitxer html ha de tenir la següent estructura

una etiqueta html i a dins dues etiquetes: header i body.
